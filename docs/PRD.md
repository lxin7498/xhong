# 基于协同过滤算法的计算机专业个性化学习资源推荐系统

## 基本信息

| 项目 | 内容 |
|------|------|
| 题目来源 | 教学模拟题目 |
| 题目类型 | 工程设计型 / 科学研究型 |
| 应用场景 | 毕设 |

## 项目定位

面向计算机专业学生，构建一个个性化学习资源推荐平台。系统通过记录用户行为（浏览、收藏、评分），利用协同过滤算法为学生推荐匹配的学习资料（视频、文章、题目），解决信息过载问题。

## 目标用户

- **主要用户**：计算机专业在校学生
- **管理员**：负责资源管理和系统维护

## 核心功能 (MVP)

### 1. 用户管理
- 注册 / 登录
- 个人信息维护
- 个人中心（查看收藏、浏览历史、评分记录）

### 2. 资源管理
- 资源类型：视频、文章、题目
- 资源元数据：标题、分类、标签、难度、来源
- 管理员：资源的增删改查

### 3. 用户行为记录
- 浏览行为记录
- 收藏 / 取消收藏
- 评分（1-5分）
- 行为日志持久化存储

### 4. 个性化推荐
- 基于用户的协同过滤（UserCF）或基于物品的协同过滤（ItemCF）
- 冷启动处理策略（新用户 / 新资源）
- 推荐列表展示（首页）

### 5. AI 学习助手（DeepSeek API）
- 右下角悬浮气泡，全局随时唤起
- 回答计算机专业学习问题（数据结构、算法、编程语言等）
- 解释推荐理由（"为什么推荐这篇资源给我？"）
- 根据学习进度给出个性化建议（"你Python学得差不多了，下一步可以看数据结构"）
- 聊天历史持久化存储

### 6. 前端页面
- 首页：推荐资源列表
- 资源详情页
- 个人中心（历史、收藏、评分）
- 登录 / 注册页
- AI 助手悬浮气泡（全局）

## 技术栈（已确认）

| 层级 | 技术 | 说明 |
|------|------|------|
| 后端 | Django | 全功能框架，ORM/Admin/认证开箱即用 |
| 前端 | Vue | 响应式框架，配合 Element Plus |
| 数据库 | MySQL | 关系型存储 |
| 推荐算法 | UserCF | 基于用户的协同过滤 |
| AI 模型 | DeepSeek API | 学习答疑、推荐解释、进度建议 |

## 数据设计要点

- **用户画像表**：基本信息 + 兴趣标签
- **资源元数据表**：标题、类型、分类、难度、链接
- **用户行为日志表**：用户ID、资源ID、行为类型（浏览/收藏/评分）、时间戳、评分值
- **AI 对话记录表**：用户ID、会话ID、消息列表、时间戳

## 推荐逻辑设计要点

- 相似度计算：余弦相似度 / 皮尔逊相关系数
- 协同过滤矩阵：用户-资源评分矩阵
- 冷启动方案：
  - 新用户 → 推荐热门/高分资源
  - 新资源 → 基于内容标签推荐给相关兴趣用户
- 推荐结果刷新策略（定时计算 vs 实时计算）

## 设计系统

采用 **Emerald Code** 设计体系，浅色默认 + 暗色可切换：

| 维度 | 规范 |
|------|------|
| 风格 | 明亮学术 + 玻璃质感（Glassmorphism），支持一键切换暗色模式 |
| 主色 | 翡翠绿 #10b981（Primary）、#006c49（深色变体） |
| 底色 | 浅灰白 #f7f9fb（Light）/ 深海军蓝 #0b1326（Dark） |
| 字体 | Space Grotesk（标题）+ Inter（正文） |
| 圆角 | 基础 4px，卡片 12px |
| 间距 | 8px 基准单位系统 |
| AI 气泡 | 右下角悬浮，脉冲动效 + 玻璃质感面板展开 |
